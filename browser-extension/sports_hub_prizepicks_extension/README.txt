# Sports Hub PrizePicks Exporter

## Install in Microsoft Edge

1. Extract this ZIP file.
2. Open Edge.
3. Go to `edge://extensions`.
4. Turn on **Developer mode**.
5. Click **Load unpacked**.
6. Select the extracted `sports_hub_prizepicks_extension` folder.

## Install in Google Chrome

1. Extract this ZIP file.
2. Open Chrome.
3. Go to `chrome://extensions`.
4. Turn on **Developer mode**.
5. Click **Load unpacked**.
6. Select the extracted folder.

## Use it

1. Open PrizePicks in the same browser.
2. Refresh the PrizePicks page after the extension is installed.
3. Wait for the board to finish loading.
4. Open the extension.
5. Leave `MLB,WNBA` in the sports box, or clear it to export all sports.
6. Click **Export CSV**.
7. Choose your Sports Hub folder:
   `C:\Users\raymo\Documents\sports-database\data\pools\incoming`

## Important

- The extension does not send a separate request to PrizePicks.
- It listens for the projection response the PrizePicks page already receives.
- PrizePicks can change its site or response format, so the extension may need updates later.
- Only install extension files you trust. This extension requests access only to PrizePicks pages, local extension storage, and downloads.
